Installasi

1. Unduh Dart SDK terbaru: Kunjungi halaman rilis Dart SDK dan unduh versi stabil terbaru yang memenuhi rentang yang diperlukan.

2. Perbarui variabel lingkungan PATH: Pastikan variabel lingkungan PATH Anda mencakup jalur ke Dart SDK.

Konfigurasi

1. Clone Proyek dari GitHub

Clone repository:

              
                git clone https://github.com/username/Aegis_pH.git

Gantilah https://github.com/username/Aegis_pH.git dengan URL repository GitHub Anda.

Masuk ke direktori proyek:

              
                cd Aegis_pH

2. Instalasi Dependensi

Instal dependensi Flutter:

              
                flutter pub get

3. Konfigurasi Proyek untuk Android

Konfigurasi file build.gradle:

              
                android {
                    compileSdkVersion 31
                    defaultConfig {
                        applicationId "com.example.aegis_ph"
                        minSdkVersion 21
                        targetSdkVersion 31
                        versionCode 1
                        versionName "1.0"
                    }
                    ...
                }

Perbarui nama paket di AndroidManifest.xml:

              
                <manifest xmlns:android="http://schemas.android.com/apk/res/android"
                    package="com.example.aegis_ph">
                    ...
                </manifest>

4. Konfigurasikan Ikon dan Splash Screen

Konfigurasikan ikon aplikasi:

Gunakan paket flutter_launcher_icons untuk mengonfigurasi ikon aplikasi.

Tambahkan dependensi di pubspec.yaml dan konfigurasikan:

              
                dev_dependencies:
                  flutter_launcher_icons: ^0.9.2

                flutter_icons:
                  android: true
                  ios: false
                  image_path: "assets/icon/icon.png"

Jalankan perintah berikut:

              
                flutter pub run flutter_launcher_icons:main

Konfigurasikan splash screen:

Gunakan paket flutter_native_splash untuk mengonfigurasi splash screen.

Tambahkan dependensi di pubspec.yaml dan konfigurasikan:

              
                dev_dependencies:
                  flutter_native_splash: ^1.2.0

                flutter_native_splash:
                  color: "#42a5f5"
                  image: assets/splash.png
                  android: true
                  ios: false

Jalankan perintah berikut:

              
                flutter pub run flutter_native_splash:create

5. Jalankan Aplikasi di Perangkat Android

Hubungkan perangkat atau buka emulator:

Hubungkan perangkat Android atau buka emulator Android.

Verifikasi perangkat:

              
                flutter devices

Jalankan aplikasi:

              
                flutter run

6. Buat APK untuk Pengguna

Buat APK untuk Android:

Untuk membuat APK, jalankan perintah berikut:

              
                flutter build apk --release

File APK akan berada di build/app/outputs/flutter-apk/app-release.apk.

Tentang Aplikasi

Aplikasi ini dirancang khusus untuk pengguna Android. Aplikasi ini digunakan untuk mendeteksi kualitas air yang layak digunakan untuk kebutuhan sehari-hari. Aplikasi ini menggunakan mikrokontroler Arduino Uno dan menggunakan sensor-sensor dari perangkat eksternal seperti DS18B20, PH-4520C, TDS Sensor Meter, dan ESP-32.

AEGIS pH

Aplikasi Pemantau Kualitas Air

Cepat & Ringan

Mulai memantau kualitas air dengan cepat di terminal! Tidak perlu lagi membuka editor teks.

Sinkronisasi File

Simpan file Anda di Dropbox dan akses dari mana saja.

Aman

Enkripsi catatan Anda opsional. Tidak ada yang bisa mengakses rahasia Anda!

Konfigurasi

Simpan semua pengaturan Anda dalam satu file config.json. Tidak perlu mengulangi pengaturan setiap kali membuat catatan.

Penyorotan

Untuk keterbacaan yang lebih baik, aegis_ph memiliki skema warna yang bersih dan indah untuk pemindaian file yang cepat.

Keybindings

Anda dapat mengharapkan keybindings umum untuk aegis_ph. Sesuaikan bindings.json sesuai selera Anda sendiri!

    Keybindings Default

  • Keluar tanpa menyimpan Ctrl+C
  • Simpan Cmd+S
  • Simpan dan Keluar Ctrl+D
  • Undo Cmd+Z

    Keybindings Markdown

  • Ctrl+A Sisipkan Tautan Markdown
  • Ctrl+I Sisipkan Gambar Markdown
  • Ctrl+V Sisipkan Video YouTube
  • Ctrl+T Sisipkan Tabel

Baca dokumentasi kami untuk keybindings lanjutan dan penyesuaian
